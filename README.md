# Introduction
A resource pack that changes the appearance of regular vanilla and modded ores to animated ores. They are emissive.

# Compatibility
- Vanilla: ✅️
- [OptiFine](https://optifine.net/downloads): ✅️
- [Continuity](https://modrinth.com/mod/continuity) (required [Indium](https://modrinth.com/mod/indium)):  ✅️
- [Connectedness](https://modrinth.com/mod/connectedness) (required [Reforgium](https://modrinth.com/mod/reforgium)): ✅️
# Embedded mods
- [Universal Ores](https://modrinth.com/mod/universal_ores): ✅
- [Deeper and Darker](https://modrinth.com/mod/deeperdarker): ✅
- [Weeping Angels](https://modrinth.com/mod/weeping-angels): ✅
# ❗❗❗ WARNING ❗❗❗ 
There may be issues with emissive of items and blocks (z-fighting) (from Minecraft **1.19.3 and later**)
### Q&A
Q: What modifications are affected by this problem? <br>
A: Only applies to Continuity + Indium, OptiFine has no problem with this. <br>

Q: How to solve the problem? <br>
A: You can work around the problem by installing [Embeddium](https://modrinth.com/mod/embeddium) (from 1.20.1 and newer), a fork of Sodium but with FRAPI support. It is also recommended to install the latest version of Fabric API (for Continuity). One note please disable Sodium and Indium before starting the game, these mods are incompatible with Embeddium.
