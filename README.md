# Introduction
A resource pack that changes the appearance of regular vanilla ores and [Universal Ores](https://modrinth.com/mod/universal_ores) to animated ores. They are emissive.

# Compatibility
- Vanilla: ✅️
- [OptiFine](https://optifine.net/downloads): ✅️
- [Continuity](https://modrinth.com/mod/continuity) (required [Indium](https://modrinth.com/mod/indium)):  ✅️
- [Universal Ores](https://modrinth.com/mod/universal_ores): ✅
